# CodeAlpha_Stock-Portfolio-Tracker
This is a simple console-based Stock Investment Tracker written in Python. The program allows users to input stock names and the number of shares they want to purchase. It uses a predefined dictionary of stock prices for several companies such as Apple, Tesla, Amazon, Nvidia, and Meta. For each valid stock input, the user is asked to enter the quantity of shares they want to buy. The program calculates the total investment by multiplying the quantity with the stock price. Input validation is implemented to ensure the user enters a valid positive integer for quantity. If the user enters an invalid value, the program prompts them again until a correct input is provided. The program also handles cases where the stock name does not exist in the dictionary. The user can type “DONE” at any time to finish the program and display the final total investment. At the end, the total investment value is saved into a text file for record keeping.
